# Product Context

**Pourquoi**
- Donner un exemple simple et robuste de “pont” *Mobile → PC → Arduino*.
- Servir de base à des projets domotiques (maison intelligente) : ampoules, relais, volets, capteurs, etc.

**Pour qui**
- Débutants Arduino / JavaScript voulant un projet concret.
- Makers voulant piloter des objets depuis un smartphone sans cloud.

**Expérience utilisateur**
- L’utilisateur ouvre l’app, appuie ON/OFF et voit la LED changer instantanément.
- En cas de problème: messages clairs (serveur arrêté, IP incorrecte, Wi‑Fi différent, port série occupé).
